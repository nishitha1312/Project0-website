# Four Elements Symbolism Website

A responsive, multi-page informational website exploring the **symbolism of the four classical elements** —  
**Fire (Agni), Earth (Prithvi), Water (Vaari), and Air (Vaayu)** — as understood in ancient philosophy.

The site presents historical, cultural, and philosophical interpretations of each element using
clean UI design and structured content.

---

## Tech Stack

- **HTML5**
- **CSS3 / SCSS**
- **Bootstrap 4**
- **Font Awesome**
- **JavaScript (Bootstrap dependencies)**

---

## Pages Included

- `index.html` – Overview of the Four Elements and their symbolism  
- `FIRE(agni).html` – Fire element symbolism  
- `EARTH(prithvi).html` – Earth element symbolism  
- `WATER(vaari).html` – Water element symbolism  
- `AIR(vaayu).html` – Air element symbolism  

---

## Features

- Responsive navigation bar using Bootstrap
- Structured semantic HTML layout
- Tabular representation of elemental qualities (hot, cold, dry, wet)
- External libraries for icons and styling
- Multi-page navigation with consistent UI
- SCSS-based styling for maintainability

---


---

## How to Run Locally

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. Navigate through the site using the menu bar

_No additional setup required._

---

## Concept Background

The website is based on ancient philosophical views that describe the physical world as being composed of four fundamental elements.  
Each element is associated with specific qualities and plays a role in maintaining balance in nature and human well-being.
